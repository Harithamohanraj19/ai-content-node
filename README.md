# AI Content Assistant (Gemini AI + Node.js + React)

**AI-powered content generation tool** built using **Google Gemini AI**, **Node.js**, and **React**.  
This app lets users generate, summarise, or rewrite text instantly, all powered by the latest LLM from Google.



**Features**

 **AI-Powered Text Generation** – Uses Gemini AI to create or rewrite content  
 **Summarization & Rewrite Modes** – Choose between *Generate*, *Summarize*, and *Rewrite*  
 **Modern React UI** – Built with Vite and styled using **Tailwind CSS**  
 **REST API with Node.js + Express** – Simple backend integration with Gemini API  


 **Frontend**
-  React (Vite)
-  Tailwind CSS
-  Axios

 **Backend**
-  Node.js + Express.js
-  Gemini AI API (Google Generative AI SDK)
-  dotenv for configuration

**Installation & Setup**

Clone the repository

`https://github.com/Harithamohanraj19/ai-content-node.git`

Install dependencies

`npm install` with react, express, axios, and dotenv

Create an inside API key in the .env file

`GEMINI_API_KEY=your_gemini_api_key_here`
